Nikon Type0016 Module SDK Revision 6
  === [Mac OS] Module SDK 使用時の注意 ===

■概要

    Macintosh版の Type0016 Module と PTP ドライバには制限事項があるため、
    Mac OS で Module SDK をご使用の際は下記にご注意ください。

■動作環境

  ※ReadMe_Jpn.txtをご参照ください。

■用語の説明

  ・新SDK ... Type0016 Module SDK Revision 6  ※本Module SDK
  ・旧SDK ... 新SDKより以前の Type0016 Module SDK

  ※その他の用語についての詳細は、ReadMe_Jpn.txtをご参照ください。

■注意事項

    ・Macintosh版の Type0016 Module には制限事項があります。
      以下のドキュメントをご参照の上、ご注意ください。

        - Usage of Type0016 Module(J).pdf
          「Macintosh版Module SDKの利用環境について」
          「Macintoshでの 使用時について」

    　旧SDK用に作成したアプリケーションから、新SDK の Type0016 Module を使用する場合も、
      上記の制限事項にご注意ください。

    ・新/旧SDK の Type0016 Module を混在させてロードしないでください。
      SDK が正常に動作しなくなります。
